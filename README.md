## oiuNewTab是什么?

chrome浏览器新标签页插件

效果图：

(https://github.com/jasonoiu/oiuNewTab/raw/master/img/demo.png)

## oiuNewTab有哪些功能？

* 所有收藏夹的访问和管理
* 最常访问链接的管理
* 博客园推荐博客和新闻显示
* 精美的当前时间模块
* 漂亮的天气预报模块
* 支持精美的壁纸切换
* 其它更丰富的功能正在努力开发中...
